# Deal Workflow by Stage

What happens at each pipeline stage and what triggers transitions.

## Prospecting
**Entry:** New lead identified (referral, inquiry, cold outreach)
**Agent tasks:**
- Log initial contact as communication
- Create client profile if new
- Research property and market comps
- Schedule intro meeting

**Exit trigger:** Client agrees to move forward → Listing

## Listing
**Entry:** Listing agreement discussion
**Agent tasks:**
- Prepare listing presentation materials
- Schedule listing appointment
- Create deal file with initial documents

**Exit trigger:** Listing agreement signed → Marketing

## Marketing
**Entry:** Property listed and active
**Agent tasks:**
- Confirm Crexi/LoopNet listing is live
- Track inquiries and showing requests
- Send weekly marketing activity reports to client
- Create OM (Offering Memorandum) if needed

**Exit trigger:** First showing scheduled → Showings

## Showings
**Entry:** Active showings in progress
**Agent tasks:**
- Coordinate showing schedules
- Send post-showing follow-ups (use template)
- Track buyer feedback
- Update client after each showing

**Exit trigger:** Buyer submits offer → Offer

## Offer
**Entry:** Written offer received
**Agent tasks:**
- Log offer details in deal
- Notify Matt immediately (approve-tier action)
- Prepare counter-offer if directed
- Track offer timeline and deadlines

**Exit trigger:** Offer accepted by both parties → Under Contract

## Under Contract
**Entry:** Fully executed purchase agreement
**Agent tasks:**
- Create closing checklist
- Track all contract deadlines (inspection period, financing, etc.)
- Coordinate with title company
- Schedule inspections

**Exit trigger:** All contingencies cleared → Due Diligence (or directly to Closing)

## Due Diligence
**Entry:** Inspection and contingency period
**Agent tasks:**
- Track inspection scheduling and results
- Coordinate repair negotiations
- Monitor financing/appraisal progress
- Update client on DD progress

**Exit trigger:** DD period expires / all items resolved → Closing

## Closing
**Entry:** Clear to close
**Agent tasks:**
- Confirm closing date and location
- Verify all documents are ready
- Send closing day reminder to all parties
- Prepare commission invoice

**Exit trigger:** Keys transferred, funds disbursed → Closed

## Closed
**Entry:** Transaction complete
**Agent tasks:**
- Send thank-you note to client
- Request testimonial/review
- Schedule 30-day check-in
- Archive deal documents
- Log final commission details
